# Contributing

Thanks for contributing!

## Quick start
```bash
python3 word_frequency.py
```

## Development
- Python 3.9+
- Optional GUI helpers: `zenity`, `python3-tk`

## Suggested workflow
1. Fork and clone the repo.
2. Create a branch for your change.
3. Keep changes focused and small.
4. Update or add tests if needed.
5. Open a PR with a clear description.

## Testing
This project has no automated tests yet. If you add any, include how to run them here.
